Intent Classification Chatbot API

About the API

This API provides a powerful intent classification chatbot that can be easily integrated into your applications. It offers several endpoints for chatting, intent detection, and accessing FAQs.

Key Features

  • Intent classification for user messages
  • Chatbot responses based on detected intents
  • Access to frequently asked questions (FAQs)
  • List of available intent categories

How to Use

To integrate this API into your application, you can make HTTP requests to the following endpoints:

POST /chat: Chat with the bot
POST /intent: Get intent of a message
GET /faqs: Get a list of FAQs
GET /intents: Get all available intent categories

Base URL: https://jayanth-mkv-chat-assist-pro.hf.space

For detailed information on request and response formats, please refer to the API documentation.

Example Usage

Here's a quick example of how to use the chat endpoint:


import requests
import json

url = "https://jayanth-mkv-chat-assist-pro.hf.space/chat"
headers = {"Content-Type": "application/json"}
data = {"message": "How do I track my order?"}

response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.json())
